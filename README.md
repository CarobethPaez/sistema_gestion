Sistema de Gestión de Datos - Python

Este proyecto consiste en un sistema de gestión de datos modular desarrollado en Python,
diseñado para automatizar procesos internos de una empresa de tecnología. Permite la
manipulación, validación y almacenamiento estructurado de información mediante estructuras
de datos avanzadas y buenas prácticas de programación.

🚀 Características Principales
● Arquitectura Modular: El código está dividido en módulos independientes (main,
gestion_datos, validaciones y utilidades) para mejorar la escalabilidad y la
mantenibilidad.
● Estructuras de Datos Eficientes:
○ Diccionarios: Almacenamiento de registros con pares clave-valor para gestionar
la información de manera estructurada.
○ Conjuntos (Sets): Garantía de integridad de datos mediante el uso de conjuntos
para evitar elementos duplicados.
○ Listas: Uso de listas para almacenar y manipular colecciones ordenadas de
datos.
○ Tuplas: Implementación de datos inmutables y organizados para elementos fijos
del sistema.

● Validación Robusta: Manejo de excepciones y validaciones de entrada para prevenir
errores de ejecución y asegurar la calidad de los datos.
● Interfaz Amigable: Uso de print() y f-strings para una salida de datos limpia,
organizada y profesional.
● Lógica Avanzada: Implementación de funciones recursivas para cálculos específicos y
modularización mediante funciones personalizadas con parámetros y retornos.
️ Tecnologías y Estándares
● Lenguaje: Python 3.13.17
● Guía de Estilo: Cumplimiento con PEP 8 para garantizar un código limpio, bien
estructurado y legible.
● Control de Versiones: Git & GitHub.

📂 Estructura del Proyecto
.
├── main.py # Punto de entrada, menú principal y estructuras de control.

├── gestion_datos.py # Lógica de manipulación de registros y funciones recursivas.
├── validaciones.py # Gestión de excepciones y validación de tipos de entrada.
├── utilidades.py # Formateo de tablas, f-strings y visualización.
└── README.md # Documentación técnica y descripción del proyecto.
